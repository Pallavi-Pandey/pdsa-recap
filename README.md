# PDSA Recap

Notes and practice problems for Programming, Data Structures, and Algorithms (PDSA).

## Project Structure

- `week1/`: Introductory week focusing on basics.
  - `ga1.ipynb`: Graded Assignments - Week 1.
  - `pa1.ipynb`: Practice Assignments - Week 1.
  - `ppa1.ipynb`: Programming Practice Assignments - Week 1.
  - `extras.md`: Additional study notes and concepts.

## Purpose
This repository is a structured recap of the PDSA curriculum, providing a centralized location for assignments and concept reviews.

## Getting Started
Open the notebooks using Jupyter to review the problems and solutions:
```bash
jupyter notebook week1/pa1.ipynb
```
